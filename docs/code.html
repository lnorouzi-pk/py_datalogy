corr_with_nulls() → pandas.core.frame.DataFrame[source]¶

See which columns have correlation with a variable being null. The rows correspond to a variable and the columns are an indicator for whether a variable is null. E.g. the data at left would result in the following table

Row | A | B | | | A_is_null | B_is_null |
|-----|—–| |---|———–|-----------|

0 | 2 | NaN | -> | A | NaN | 1 | 1 | 1 | 3.7 | | B | 0 | NaN | 2 | 2 | NaN | 3 | NaN | 3.8 | 4 | 1 | 3.8 | 5 | NaN | 3.7 |

When calculating correlation, rows are omitted if the variable is null in a pairwise fashion. E.g. for calculating correlation between A and the indicator B_is_null, rows 3 and 5 are omitted. Only numeric columns are included as rows, and only columns with nulls are included as columns.

Parameters:
  • visual (bool.) – Whether draw the results or not. Default is True to save pictures.
  • save_pic (bool.) – Whether save the results or not. Default is True to save pictures.
Returns:

correlations: pd.DataFrame – Correlations as described above
